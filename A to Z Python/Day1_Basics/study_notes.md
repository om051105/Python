# Day 1: Basics & Syntax - Quick Recap

## 1. Variables & Types
- `name = "Antigravity"` (String)
- `age = 25` (Integer)
- `price = 19.99` (Float)
- `is_learning = True` (Boolean)

## 2. Input/Output
- `print("Hello", name)` - Outputting data.
- `user_input = input("Enter something: ")` - Getting data (always returns a string).

## 3. Type Casting
- `int("5")` -> Converts string to integer.
- `str(10)` -> Converts integer to string.

## 4. Basic Math
`+`, `-`, `*`, `/`, `//` (floor division), `%` (modulus), `**` (exponent).
